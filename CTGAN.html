CTGAN Model Overview

The CTGAN model (Conditional Tabular GAN) generates synthetic tabular data to enable safe data sharing and robust testing.

Applications:

  • Synthetic tabular data generation
  • Improving machine learning robustness
  • Ensuring privacy compliance
